# GeneCards

## What is GeneCards?

GeneCards is a comprehensive human gene database that provides detailed information about genes, proteins, diseases, pathways, and biological functions.

---

## Website

https://www.genecards.org/

---

## Used For

- Gene search
- Disease association
- Gene function
- Expression data
- Protein information

---
